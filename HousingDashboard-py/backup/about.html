About LiveTwiceByThinkingTwice

LiveTwiceByThinkingTwiceg is a student-developed web platform created to simplify housing decisions in Sheffield. Whether you're a university student, local resident, or newcomer to the city, LiveTwiceByThinkingTwiceg provides reliable, visual insights into neighborhood affordability, livability, and access to green spaces.

The project was built using HTML, CSS, JavaScript, and Leaflet.js, leveraging open data such as GeoJSON and local statistics. Our interactive map and filter system allow users to compare neighborhoods based on cost, green space access, and overall livability score.

LiveTwiceByThinkingTwiceg was developed as part of a software engineering module to demonstrate practical skills in web development, data visualization, and user-focused design. It reflects our core mission: to help people live smarter, more informed lives by thinking twice about where they live.

  • 🧠 Data-informed neighborhood comparisons
  • 🌳 Green space visual filters
  • 💷 Rental cost sliders for budgeting

Thank you for exploring our project — we hope it helps you find the perfect place to live in Sheffield.
